# Spotee（スポッティー）

> すべての人が、次の"行きたい場所"に出会えるサービス

## 技術スタック

- **Frontend**: Next.js 14 (App Router), Apollo Client, Tailwind CSS
- **Backend**: NestJS, Apollo Server (GraphQL), Prisma
- **Database**: PostgreSQL (Supabase)
- **Auth**: Supabase Auth
- **Hosting**: Vercel (Frontend), Railway (Backend)

## 開発環境のセットアップ

```bash
# 依存関係のインストール
npm install

# フロントエンド開発サーバー起動
npm run dev:frontend

# バックエンド開発サーバー起動
npm run dev:backend
